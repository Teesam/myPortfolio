Hyper Text Mark up Language (HTML)

The very first technology you'll most likely get your hands on if you're looking to get onto frontend web dev. The magic HTML 5 produces is profounding, giving you an alomost seamless effort in populating the web.

Cascading Style Sheet (CSS)

Have you ever taken the time out to wonder about whatever it is that's responsible for making the web pages you visit every other day look so beautiful? Puzzle no more. CSS does so much under the hood than is given credit for. HTML 5 + CSS = beautiful.

Javascript

Into the world of programming language. This guy here called javascript, is the what you'd call the king of frontend. It's considered one of the most popular/well used programming languages in the world if not beyond. It provides a limitless world of possibilities. If you can think it, you can get it done.

React Js

Let's talk React. Arguably the most popular javascript framework as it stands. Allows for possibilities that might not easily be achieved by using vanilla javascript.

Vue Js

Some would say it's the baby of the house after Angular and React, but there is a reason it has gained popularity in recent years. Learning scope, documentation and so much more are beyond applaudable.
